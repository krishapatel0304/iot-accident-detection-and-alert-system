# IoT Obstacle Detection Alert System

## Project Description
This is a basic IoT project that helps drivers detect nearby obstacles using an ultrasonic sensor. 
When an object or vehicle comes too close, a buzzer alerts the driver.

This system can help prevent accidents by warning the driver before a collision occurs.

## Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Buzzer
- Breadboard
- Jumper Wires

## Working Principle
The ultrasonic sensor measures the distance between the vehicle and nearby obstacles.

If the distance becomes smaller than a predefined limit, the buzzer turns ON and alerts the driver.

If the distance is safe, the buzzer remains OFF.

## Applications
- Vehicle safety system
- Obstacle detection
- Smart parking assistance
